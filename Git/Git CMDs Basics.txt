Skipping files:
  1. touch .gitignore
  2. Add filename per line
  
Adding a file to staging area
  git add <filename>

Adding all files to staging area
  git add -A

Resetting staging area
  git reset
  
Remove specific additions
  git reset <filename>
 
Commit changes
  git commit -m "Msg"
  
View commit log
  git log
